Tech Pulse Documentation
========================

**Author:** Matanda Software

**Version:** 1.0.0

**GitHub:** https://github.com/matandasoftware/tech-pulse

Tech Pulse is a Django REST API for aggregating and managing tech news articles from multiple RSS sources with advanced filtering, search, and automation capabilities.
